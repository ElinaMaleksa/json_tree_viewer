## Features

- Pretty view the json data;
- Custom the json element color;
- Support copy key or value text;

## Getting started

```yaml
dependencies:
    json_tree_viewer: ^0.0.2
```

## Usage

```dart
import 'package:json_tree_viewer/json_tree_viewer.dart';

JsonTreeViewer(
    data: {
    	"name": "Beijing",
        "country": "China",
        "whereami": false,
    },
)
```
